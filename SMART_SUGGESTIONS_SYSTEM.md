# نظام الاقتراحات الذكية والأذونات الاحترافي

## 📋 نظرة عامة

تم تطوير نظام متكامل للاقتراحات الذكية المترابطة ونظام أذونات احترافي يشرح كل إذن بالتفصيل.

---

## 🎯 الميزات الرئيسية

### 1. نظام الأذونات الاحترافي
#### ✨ ما تم تنفيذه:
- **شاشة ترحيب تفاعلية** مع 3 صفحات:
  - صفحة الترحيب مع ميزات التطبيق
  - صفحة الأذونات مع شرح تفصيلي لكل إذن
  - صفحة الإكمال

- **شرح مفصل لكل إذن**:
  - **إشعارات الصلاة**: للحصول على تنبيهات أوقات الصلاة والأذكار اليومية
  - **التنبيهات الدقيقة**: لضمان وصول الأذان في الوقت المحدد بالضبط
  - **تجاوز وضع عدم الإزعاج**: لإظهار الأذان حتى في الوضع الصامت
  - **الموقع الجغرافي**: لحساب مواقيت الصلاة الدقيقة واتجاه القبلة
  - **عرض ملء الشاشة**: لإظهار الأذان بشكل واضح حتى عند قفل الشاشة

- **تصنيف الأذونات**:
  - أذونات أساسية (مطلوبة)
  - أذونات اختيارية (موصى بها)

- **تجربة مستخدم احترافية**:
  - شرح الفائدة من كل إذن
  - إمكانية التخطي
  - مؤشر تقدم واضح
  - تصميم جذاب مع أيقونات ملونة

### 2. محرك الاقتراحات الذكية 🤖

#### الترابط الزمني (Time-Based)
✅ **بعد أذان الفجر** (5:00-9:00):
- اقتراح أذكار الصباح
- تذكير بصلاة الإشراق (بعد 20 دقيقة من الشروق)
- عرض حديث قصير عن فضل الذكر

✅ **وقت الضحى** (9:00-11:30):
- اقتراح صلاة الضحى
- "يصبح على كل سُلامى من أحدكم صدقة"

✅ **بعد المغرب** (17:00-20:00):
- أذكار المساء
- اقتراح قراءة جزء محدد من الورد

✅ **بعد العشاء** (21:00-2:00):
- تذكير بالوتر
- عرض آيات القيام (آخر المزمل / آخر البقرة)

✅ **قيام الليل** (2:00-5:00):
- اقتراح قيام الليل في الثلث الأخير
- "ينزل ربنا تبارك وتعالى كل ليلة إلى السماء الدنيا"

#### الترابط الأسبوعي
✅ **يوم الجمعة صباحًا**:
- اقتراح قراءة سورة الكهف
- "نور من الجمعة إلى الجمعة"

✅ **ساعة الإجابة يوم الجمعة** (14:00-18:00):
- تنبيه بساعة الإجابة
- اقتراح الإكثار من الدعاء والصلاة على النبي ﷺ
- أولوية عالية جداً (Critical)

✅ **ليلة الجمعة** (الخميس مساءً):
- تكثيف الصلاة على النبي ﷺ
- "أكثروا الصلاة علي يوم الجمعة وليلة الجمعة"

#### الترابط الموسمي
✅ **شهر رمضان**:
- اقتراح ختمة رمضان المنظمة
- واجهة خاصة بشهر الخير

✅ **العشر الأواخر من رمضان**:
- اقتراح دعاء ليلة القدر
- "اللهم إنك عفو تحب العفو فاعف عني"
- تذكير بقيام الليل

✅ **الأيام البيض** (13، 14، 15 هجري):
- تنبيه بصيام الأيام البيض
- دعاء الصائم
- فضل صيام ثلاثة أيام من كل شهر

#### الترابط السلوكي (Behavior-Based)
✅ **إذا توقف المستخدم عن الورد 3 أيام**:
- إشعار هادئ "استئناف من حيث توقفت"
- عرض عدد الأيام المتوقفة

✅ **إذا أكمل ختمة**:
- اقتراح دعاء ختم القرآن
- إنشاء ختمة جديدة تلقائيًا

✅ **إذا قرأ سورة معينة**:
- اقتراح قراءة تفسيرها
- اقتراح سماعها بصوت قارئ آخر

✅ **إذا لم تكن له ختمة نشطة**:
- اقتراح إنشاء ختمة جديدة
- "نظّم قراءتك للقرآن"

#### الترابط بالأحداث (Event-Based)
✅ **بعد انتهاء الصلاة**:
- أذكار بعد الصلاة
- تسبيح 33 + 33 + 34 تلقائيًا
- آية الكرسي

✅ **بعد التسبيح 100 مرة**:
- عرض فضل الذكر المذكور
- "من سبّح الله دبر كل صلاة..."

✅ **بعد إكمال ختمة**:
- دعاء ختم القرآن
- احتفال بالإنجاز

✅ **بعد قراءة حديث**:
- اقتراح آية مرتبطة بالموضوع
- اقتراح شرح مختصر

✅ **عند دخول شاشة القبلة**:
- دعاء التوجه للقبلة
- "وجهت وجهي للذي فطر السماوات والأرض..."

## 🚀 كيفية الاستخدام

### 1. عرض شاشة الأذونات عند التشغيل الأول

```dart
import 'package:flutter/material.dart';
import 'core/permissions/screens/onboarding_permissions_screen.dart';

// في main.dart أو app_router.dart
OnboardingPermissionsScreen(
  onComplete: () {
    // الانتقال للشاشة الرئيسية
    Navigator.pushReplacement(
      context,
      MaterialPageRoute(builder: (_) => HomeScreen()),
    );
  },
)
```

### 2. عرض الاقتراحات في الشاشة الرئيسية

```dart
import 'package:flutter_riverpod/flutter_riverpod.dart';
import 'core/suggestions/providers/suggestions_provider.dart';
import 'core/suggestions/widgets/suggestion_card.dart';

class HomeScreen extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final suggestions = ref.watch(prioritySuggestionsProvider); // فقط الأولوية العالية

    return Column(
      children: [
        // عرض الاقتراحات
        if (suggestions.isNotEmpty)
          ...suggestions.take(2).map((suggestion) =>
            CompactSuggestionCard(
              suggestion: suggestion,
              onDismiss: () {
                ref.read(suggestionsProvider.notifier)
                   .dismissSuggestion(suggestion.id);
              },
            ),
          ),

        // باقي محتوى الشاشة...
      ],
    );
  }
}
```

```dart

final enableSeasonalSuggestionsProvider = ...

```

### 2. تعديل أوقات الاقتراحات

يمكن تعديل الأوقات في `suggestions_engine.dart`:

```dart
// مثال: تغيير وقت أذكار الصباح
if (context.hourOfDay >= 5 && context.hourOfDay < 10) { // من 5 إلى 10 صباحاً
  // ...
}
```

### 3. إضافة اقتراحات جديدة

في `suggestions_engine.dart`، أضف في الدالة المناسبة:

```dart

  // أضف اقتراحاتك الجديدة هنا

    Suggestion(
      id: 'my_new_suggestion',

      // ...
```

## 🎯 أفضل الممارسات

### 1. لا تكثر من الاقتراحات
- اعرض 1-3 اقتراحات كحد أقصى في الشاشة الرئيسية
- استخدم `prioritySuggestionsProvider` للاقتراحات المهمة فقط

### 2. احترم خصوصية المستخدم
- جميع البيانات محفوظة محلياً
- لا ترسل أي بيانات للخوادم

### 3. اجعل الاقتراحات قابلة للرفض
- دائماً اسمح للمستخدم بإخفاء الاقتراح
- لا تعيد عرض الاقتراحات المرفوضة

### 4. اختبر الأوقات
- تأكد من دقة أوقات الاقتراحات
- اختبر في أوقات مختلفة من اليوم والأسبوع

## 🐛 المشاكل الشائعة والحلول

### المشكلة: الاقتراحات لا تظهر
**الحل**: تأكد من:
1. استدعاء `refresh()` بعد التسجيل في الداخل
2. عدم رفض الاقتراح مسبقاً
3. الوقت الحالي يتوافق مع شروط الاقتراح

### المشكلة: الاقتراحات تظهر في أوقات خاطئة
**الحل**: راجع منطق الأوقات في `_getTimeBasedSuggestions()`

### المشكلة: الاقتراح لا يختفي بعد انتهاء وقته
**الحل**: تأكد من تحديد `expiresAt` بشكل صحيح

## 🚀 التطوير المستقبلي

4. 🔄 إشعارات push للاقتراحات المهمة
5. 🔄 إحصائيات عن الاقتراحات (ما الأكثر تفاعلاً؟)
6. 🔄 مشاركة الاقتراحات مع الأصدقاء

## 📝 ملاحظات تقنية

### الأداء
- الاقتراحات تُحسب عند الطلب (on-demand)
- استخدم `select` في Riverpod لتقليل إعادة البناء:
  ```dart
  final count = ref.watch(suggestionsCountProvider);
  ```

### الذاكرة
- الاقتراحات المرفوضة محفوظة في SharedPreferences
- يمكن مسحها يدوياً عبر `clearDismissed()`

### التوافق
- يعمل على Android و iOS
- بعض الميزات تتطلب أذونات خاصة (مثل تجاوز DND)
