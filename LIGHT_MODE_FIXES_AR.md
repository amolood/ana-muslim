### 1. إنشاء نظام الألوان الدلالية 🎨

تم إنشاء ملف جديد: [`lib/core/theme/app_semantic_colors.dart`](lib/core/theme/app_semantic_colors.dart)

**المميزات:**

```dart
// ألوان الحدود

// ألوان الأيقونات
iconPrimary      // أيقونة أساسية

// ألوان الحالة
```

```dart
import 'package:im_muslim/core/theme/app_semantic_colors.dart';

Text(
  'صلاة الفجر',
  style: TextStyle(color: context.colors.textPrimary),
)
```

### الطريقة 2: استخدام TextTheme (الأفضل للنصوص)
```dart
Text('مواقيت الصلاة', style: Theme.of(context).textTheme.titleLarge)

```

```dart
Container(
  color: Theme.of(context).colorScheme.surfaceContainerHighest,

    'المحتوى',
    style: TextStyle(color: Theme.of(context).colorScheme.onSurface),
  ),
)
```

## ✨ نمط الاستبدال السريع

| ❌ استبدل هذا | ✅ بهذا |
|----------------|---------|
| `Colors.white` | `Theme.of(context).colorScheme.onSurface` أو `context.colors.textPrimary` |
| `Colors.grey` | `context.colors.textSecondary` أو `.textTertiary` |
| `Colors.white70` | `context.colors.textSecondary` (بدون شفافية!) |
| `.withOpacity(0.X)` | استخدم الرموز الدلالية بدلاً من ذلك |
| `AppColors.surface(context)` | `Theme.of(context).colorScheme.surfaceContainerHighest` |

## ✅ قائمة الفحص للاختبار

قبل اعتبار الشاشة "مكتملة":

- [ ] جميع النصوص مقروءة في الوضع النهاري (14pt كحد أدنى)
- [ ] لا توجد ألوان مشفرة: `Colors.white`, `Colors.black`, `Colors.grey`
- [ ] لا يوجد `.withOpacity()` على ألوان النصوص/الأيقونات
- [ ] نسبة تباين ≥ 4.5:1 للنص العادي
- [ ] نسبة تباين ≥ 3:1 للنص الكبير (18pt+)
- [ ] الوضع الليلي يعمل تمامًا
- [ ] النص العربي واضح
- [ ] النص الإنجليزي واضح
- [ ] الحالات المعطلة واضحة
- [ ] حالات الأخطاء واضحة
- [ ] البطاقات لها حدود/ظلال مناسبة
- [ ] حقول الإدخال لها تسميات وتلميحات واضحة
